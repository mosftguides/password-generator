Генератор паролей

Простой в использовании, быстрый и подходящий для всех устройств генератор паролей!


Документация по API

Если вы хотите использовать API генератора паролей, вам необходимо сделать следующее:

POST api.php [len] [qua] [son]

Параметр

Параметр  значение
len Параметр "len" указывает длину пароля. Значение может быть любым числом без десятичных знаков, которое больше 1.
qua Параметр qua (сокращение от quanity) обозначает количество паролей.
Значением может быть любое число без десятичных знаков, превышающее 1.
son Параметр "сын", сокращение от специальных символов, указывает, следует ли использовать специальные символы.
Значение может быть либо 0 (не использовать специальные символы), либо 1 (использовать специальные символы).
